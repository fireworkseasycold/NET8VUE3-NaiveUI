# 系统架构设计 v1.0

## 1. 总体架构

系统包含 四个业务子系统：

电商（Ecommerce）

工业 IoT（Industrial IoT）

AI SaaS（AI 服务平台）

工具平台（Tools Platform）

每个业务 = 前端（Vue3 + Naive UI） + 后端（.NET8 + SqlSugar） + MySQL，统一由 Docker Compose 管理。

2. 模块关系
